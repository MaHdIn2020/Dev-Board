Question-1: What are the different ways to select an element in the DOM?

Use getElementById, getElementsByClassName, getElementsByTagName, querySelector, or querySelectorAll to select elements by ID, class, tag, or CSS selector.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML includes HTML tags, innerText shows visible text, and textContent shows all text

Question-3: What is event delegation in the DOM?

Event delegation adds an event listener to a parent element to handle events for its children, useful for dynamic content.

Question-4: What is event bubbling in the DOM?

Event bubbling is when an event starts at the target element and propagates up through its ancestors in the DOM tree.

Question-5: How do you create, add, and remove elements using JavaScript?

Create with createElement, add with appendChild, and remove with removeChild or remove.
